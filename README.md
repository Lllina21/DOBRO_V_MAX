# Добро в MAX

Мини-приложение для волонтёрства и благотворительности в мессенджере MAX.

## Описание

Платформа объединяет волонтёров и социальные организации, позволяя:
- Организациям находить волонтёров для своих проектов
- Волонтёрам находить подходящие возможности помочь
- Нуждающимся создавать заявки на помощь
- Всем участникам безопасно общаться через встроенный чат

## Технологии

- **React 18** - UI библиотека
- **React Router** - маршрутизация
- **Vite** - сборщик и dev-сервер
- **MAX UI** - компоненты дизайн-системы MAX
- **MAX Bridge** - интеграция с мессенджером MAX

## Установка и запуск

### Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build

# Просмотр production сборки
npm run preview
```

### Docker

```bash
# Сборка образа
docker build -t dobro-v-max-frontend .

# Запуск контейнера
docker run -p 80:80 dobro-v-max-frontend
```

## Функционал

### Для организаций
- Регистрация и верификация
- Публикация заявок на помощь
- Управление заявками
- Общение с волонтёрами

### Для волонтёров
- Просмотр каталога заявок
- Фильтрация по регионам, категориям, типам
- Отклик на заявки
- Общение с организациями

### Для нуждающихся
- Создание заявок на помощь
- Автоматический подбор организаций
- Получение уведомлений

### Коммуникация
- Встроенный чат для безопасного общения
- Уведомления о новых сообщениях
- История переписки

## Интеграция с MAX

Приложение использует MAX Bridge для:
- Получения информации о пользователе
- Вызова стандартных действий MAX
- Адаптации под дизайн-систему мессенджера

## Адаптивность

Приложение адаптировано для:
- Больших экранов (от 1440px) - приоритет
- Планшетов (768px - 1440px)
- Мобильных устройств (до 768px)
