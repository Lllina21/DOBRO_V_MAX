# Отладка приложения

## Как проверить, что всё работает:

1. **Откройте консоль браузера** (F12 → Console)

2. **Проверьте загрузку данных:**
   - В консоли должны быть сообщения о создании заявок
   - Проверьте localStorage: в консоли выполните:
     ```javascript
     JSON.parse(localStorage.getItem('dobro_requests'))
     ```

3. **Проверьте создание заявки:**
   - Перейдите в "Создать заявку"
   - Заполните форму
   - Нажмите "Создать заявку"
   - В консоли должно появиться: "Создание заявки: ..."
   - После этого должно быть: "Обновлённый список заявок: ..."

4. **Проверьте каталог:**
   - Перейдите в "Каталог"
   - Должны отображаться заявки
   - Если пусто, проверьте консоль на ошибки

5. **Очистка данных (если нужно):**
   ```javascript
   localStorage.clear()
   location.reload()
   ```

## Возможные проблемы:

- **Заявки не отображаются:** Проверьте, что данные загружаются из localStorage
- **Формы не работают:** Проверьте консоль на ошибки JavaScript
- **Чат не открывается:** Убедитесь, что отклик создаётся успешно
